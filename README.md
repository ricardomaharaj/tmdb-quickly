## TMDB Quickly

Frontend for The Movie Database written in React and Typescript

### Features

- Graphql with caching and minimal data fetching
  - Designed to work well in slow data environments
- Client side routing with URL query variables and parameters
  - shareable links that correctly load the exact same page and state
- PWA support for automatic site caching and instant loading upon subsequent visits
- React router relative routing and search params replacement
